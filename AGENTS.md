# Default Agent Instructions

## UI/UX Default Rule
- For any UI, frontend, styling, layout, visual, component, page, dashboard, or UX-related request in this repository, always use the `ui-ux-pro-max` skill by default.
- Follow the skill workflow, including running its required `--design-system` step before implementation.
- Preserve the existing project visual language unless the user explicitly asks for a new style direction.
